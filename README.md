# Leitor de Arquivo TXT em Java

Projeto de ler um arquivo em TXT simples em Java, utilizando POO
## Status
Concluído

## Funcionalidades
- Realiza a leitura de um arquivo TXT
- Indica o numero de linhas totais do arquivo
- Trata erros de entrada inválida

## Tecnologias utilizadas
- Java
- IntelliJ IDEA
- Maven
- Git e Github


## Como executar
1. Instale o Java(versão 17 ou superior)
2. Clone ou baixe este repositório:  
   git clone https://github.com/CaioAraujoDev/Leitor-de-Arquivo-TXT-em-Java.git
3. Abra o projeto no seu IDE(recomendado: Intellij IDEA)
4. Execute a classe principal(Main)
## Exemplo de uso
Primeiro adicione seu Arquivo TXT dentro da pasta resource(src/main/resources)  
Em seguida, coloque o nome: teste
